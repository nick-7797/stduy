🐍 4일차: 문자열 자유자재로 다루기

문자열의 특정 글자를 콕 집어내거나, 원하는 부분만 잘라내는 법을 배워요.

오늘의 주요 개념 💡

문자열은 글자들이 순서대로 나열된 기차와 같아요. 우리는 각 칸에 번호를 매겨 원하는 칸의 글자를 꺼내보거나, 여러 칸을 한 번에 잘라낼 수 있답니다.

1. 인덱싱 (Indexing): 한 글자 콕 집어내기

문자열의 각 글자에는 순서대로 번호가 붙어있어요. 이 번호를 '인덱스'라고 부릅니다.

  • 컴퓨터는 숫자를 0부터 세기 시작해요! 첫 번째 글자의 인덱스는 0입니다.
  • 변수명[번호] 형식으로 특정 위치의 글자를 가져올 수 있어요.
  • 마이너스 인덱스: -1은 맨 마지막 글자, -2는 뒤에서 두 번째 글자를 의미해요.

2. 슬라이싱 (Slicing): 원하는 만큼 잘라내기

문자열의 특정 부분을 잘라내고 싶을 때 사용해요.

  • 변수명[시작번호:끝번호] 형식으로 사용해요.
  • 🚨 중요! 끝번호에 해당하는 글자는 포함되지 않아요. (끝번호 바로 앞까지 잘라내요)
  • [:끝번호]: 처음부터 끝번호 앞까지 잘라내요.
  • [시작번호:]: 시작번호부터 끝까지 잘라내요.

✍️ 예제 살펴보기

"Life is too short" 라는 문장에서 첫 글자, 마지막 글자, 그리고 'Life'와 'short' 단어를 각각 뽑아내 볼게요.

sentence = "Life is too short" # 인덱싱 예제 first_char = sentence[0] # 첫 번째 글자 (0번 인덱스) last_char = sentence[-1] # 마지막 글자 (-1번 인덱스) # 슬라이싱 예제 word_life = sentence[0:4] # 0번부터 4번 앞까지 (0,1,2,3) word_short = sentence[-5:] # 뒤에서 5번째부터 끝까지 print("첫 글자:", first_char) print("마지막 글자:", last_char) print("첫 단어:", word_life) print("마지막 단어:", word_short)

💬 예제 설명

sentence[0]은 'L'을, sentence[-1]은 't'를 정확히 집어냈죠. sentence[0:4]를 보세요. 4번 인덱스인 ' ' (공백)은 포함되지 않고, 0, 1, 2, 3번 인덱스에 해당하는 'Life'만 깔끔하게 잘라냈어요. sentence[-5:]처럼 시작번호만 지정하면 끝까지 모두 잘라낼 수 있답니다.

사용자에게 질문하고, 답변을 받아 활용하는 법을 배워봐요!

오늘의 주요 개념 💡

어제는 우리가 파이썬에게 일방적으로 말을 걸었죠? 오늘은 반대로 파이썬이 우리의 말을 듣게 하는 방법을 배울 거예요. 프로그램을 훨씬 더 유용하게 만들어 준답니다.

1. `input()` 함수: 사용자의 목소리를 듣는 귀

input() 함수는 프로그램이 잠시 멈추고 사용자가 키보드로 무언가를 입력할 때까지 기다려주는 역할을 해요. 사용자가 입력한 내용은 변수에 저장해서 나중에 다시 쓸 수 있죠.

🚨 아주 중요해요! input()으로 받은 정보는 숫자를 입력해도 무조건 문자열(String)로 취급돼요. 계산을 하려면 꼭 변신시켜 줘야 해요!

2. 자료형 변환 (Type Casting): 정보의 옷 갈아입히기

문자열 "25"와 숫자 10은 바로 더할 수 없어요. 이럴 때 정보의 종류(자료형)를 바꿔주는 '자료형 변환'이 필요해요.

  • int(): 문자열을 정수(Integer)로 변신! 예: int("25") → 숫자 25
  • float(): 문자열을 실수(Float)로 변신! 예: float("3.14") → 숫자 3.14
  • str(): 숫자를 다시 문자열(String)로 변신! 예: str(100) → 글자 "100"

✍️ 예제 살펴보기

사용자에게 이름과 나이를 물어보고, 10년 뒤 나이를 알려주는 프로그램을 함께 보시죠!

# input() 함수로 사용자에게 질문하고 답변을 변수에 저장해요. user_name = input("당신의 이름은 무엇인가요? ") age_str = input("올해 나이가 어떻게 되세요? ") # 입력받은 나이(문자열)를 계산할 수 있도록 정수(int)로 변환해요. user_age = int(age_str) # 10년 뒤 나이를 계산해요. future_age = user_age + 10 # 최종 결과를 출력해요. print(user_name, "님, 안녕하세요!") print("10년 뒤에는", future_age, "살이 되시는군요!")

💬 예제 설명

age_str = input(...) 코드는 사용자에게 나이를 물어보고, 그 답변을 age_str 변수에 **문자열**로 저장해요. 바로 다음 줄 user_age = int(age_str)에서 int() 함수가 이 문자열을 계산 가능한 **숫자**로 변신시켜 주는 것이 핵심이랍니다! 이 과정을 거쳐야 user_age + 10 같은 계산이 가능해져요.

🐍 3일차: 계산의 달인 되기

파이썬의 다양한 계산 기능을 마스터해 봐요!

오늘의 주요 개념 💡

파이썬은 아주 똑똑한 계산기예요. 단순히 더하고 빼는 것뿐만 아니라, 몫만 구하거나 나머지만 구하는 특별한 계산도 할 수 있답니다.

+ : 더하기
- : 빼기
* : 곱하기
/ : 나누기 (결과가 소수점으로 나와요)

✨ 특별한 계산 친구들 ✨

// : 몫 구하기 (예: 7 // 3 → 2)
% : 나머지 구하기 (예: 7 % 3 → 1)
** : 거듭제곱 (예: 2 ** 3 → 8)

✍️ 예제 살펴보기

사탕 20개를 3명의 친구에게 똑같이 나눠주고 남은 사탕은 내가 가지려고 해요. 한 명당 몇 개를 갖고, 내겐 몇 개가 남을까요?

total_candies = 20 num_friends = 3 # 한 명당 나눠줄 사탕의 개수 (몫) candies_per_friend = total_candies // num_friends # 나눠주고 남은 사탕의 개수 (나머지) remaining_candies = total_candies % num_friends print("총 사탕:", total_candies, "개") print("친구 수:", num_friends, "명") print("한 명당", candies_per_friend, "개씩 가질 수 있어요.") print("그리고 저에게는", remaining_candies, "개가 남아요!")

💬 예제 설명

여기서 핵심은 //와 % 연산자예요. 그냥 /로 나누면 6.666... 같은 소수점이 나오지만, //를 쓰면 딱 떨어지는 몫인 6만 얻을 수 있죠. 그리고 %를 사용하면 나눠주고 남은 나머지인 2를 정확히 계산할 수 있답니다. 정말 편리하죠?

🐍 1일차: 파이썬과의 첫 만남

컴퓨터에게 인사를 건네고, 정보를 저장하는 법을 배워봐요!

오늘의 주요 개념 💡

오늘은 프로그래밍의 가장 기본이 되는 세 가지를 배울 거예요. 바로 컴퓨터 화면에 글자를 보여주는 것, 정보를 잠시 담아두는 상자를 만드는 것, 그리고 그 정보의 종류를 알아보는 것이랍니다.

1. `print()` 함수: 컴퓨터와 대화하기

print()는 파이썬에게 "이것 좀 화면에 보여줘!"라고 말하는 것과 같아요. 괄호 () 안에 우리가 보여주고 싶은 내용을 넣으면, 컴퓨터가 화면에 그대로 출력해준답니다. 프로그래밍의 가장 기본적이면서도 중요한 친구죠.

2. 변수 (Variable): 정보를 담는 상자

이름, 나이, 날짜 같은 정보를 저장했다가 나중에 다시 쓰고 싶을 때 사용하는 '이름표가 붙은 상자'예요. 등호(=)를 사용해서 my_name = "김파이"처럼 상자에 정보를 담을 수 있어요.

3. 자료형 (Data Types): 정보의 종류

변수 상자에 담는 정보의 종류를 말해요. 오늘은 세 가지만 기억해 주세요!

  • 문자열 (String): "안녕하세요!" 처럼 따옴표로 감싸인 글자들.
  • 정수 (Integer): 1, 100, -5 처럼 소수점이 없는 숫자.
  • 실수 (Float): 3.14, -0.5 처럼 소수점이 있는 숫자.

✍️ 예제 살펴보기

백문이 불여일견! 코드로 직접 확인해 볼까요? 아래 코드는 이름, 나이, 키를 각각의 변수에 저장하고 화면에 자기소개를 출력하는 예제예요.

# 자기소개를 위한 변수들을 만들어 봐요. my_name = "김파이" # 이름 (문자열) my_age = 25 # 나이 (정수) my_height = 175.5 # 키 (실수) # print() 함수를 사용해서 자기소개를 출력해 봐요. print("안녕하세요!") print("제 이름은", my_name, "입니다.") print("나이는", my_age, "살이고, 키는", my_height, "cm입니다.")

💬 예제 설명

코드가 어떻게 작동하는지 궁금하셨죠? my_name = "김파이"는 'my_name'이라는 상자에 "김파이"라는 글자를 담는 과정이에요. 그리고 print("제 이름은", my_name, "입니다.")는 "제 이름은"이라는 글자와 my_name 상자 안의 내용("김파이"), 그리고 "입니다."라는 글자를 합쳐서 화면에 보여달라는 뜻이랍니다. 정말 간단하죠?

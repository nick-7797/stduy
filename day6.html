조건문을 사용해 프로그램이 스스로 판단하고 결정하게 만들어 봐요!

오늘의 주요 개념 💡

"만약 비가 오면 우산을 챙기고, 그게 아니라면 그냥 나간다"처럼, 우리도 일상에서 조건에 따라 행동을 결정하죠. 파이썬에게도 이런 판단 능력을 줄 수 있어요.

1. `if`, `elif`, `else`: 조건에 따른 선택지

  • if 조건:: "만약 이 조건이 맞다면, 아래 코드를 실행해!" 라는 뜻이에요.
  • elif 조건:: "위의 조건이 틀리고, 만약 이 조건이 맞다면, 이걸 실행해!" (여러 개 사용 가능)
  • else:: "위의 조건들이 모두 틀리다면, 마지막으로 이걸 실행해!" (선택사항, 맨 마지막에 한 번만)

🚨 문법 약속! 조건문 뒤에는 반드시 콜론(:)을 찍고, 다음 줄부터는 꼭 들여쓰기(indentation)를 해야 해요. 들여쓰기 된 코드 블록이 바로 해당 조건이 맞을 때 실행되는 영역이랍니다.

2. 비교 연산자: 조건을 만드는 도구

조건을 만들 때는 아래와 같은 비교 연산자를 사용해요.

  • == : 같다 (=는 변수에 값을 넣는 것이니 헷갈리면 안 돼요!)
  • != : 다르다
  • > : 크다
  • < : 작다
  • >= : 크거나 같다
  • <= : 작거나 같다

✍️ 예제 살펴보기

나이를 입력받아 놀이기구 탑승 가능 여부를 알려주는 프로그램이에요.

age_str = input("나이를 입력하세요: ") age = int(age_str) if age >= 10: print("환영합니다! 놀이기구를 즐겨주세요.") else: print("아쉽지만, 10살 이상부터 탑승 가능합니다.") remaining_years = 10 - age print(remaining_years, "년 뒤에 다시 만나요!")

💬 예제 설명

먼저 나이를 입력받아 숫자(int)로 바꿔줘요. 그리고 if age >= 10: 이라는 조건으로 "나이가 10보다 크거나 같은가?"를 확인하죠. 이 조건이 맞으면(True) 첫 번째 `print` 문이 실행되고, 조건이 틀리면(False) else: 아래의 두 `print` 문이 실행됩니다. 프로그램이 상황에 맞게 다른 메시지를 보여주죠!
